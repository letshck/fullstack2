Single Page Application (SPA) using React
Project Overview

This project focuses on understanding and implementing a modern Single Page Application (SPA) using React. It demonstrates the core concepts of frontend development including component-based architecture, client-side routing, state management, API integration, and deployment.

The application is built using Node.js, npm, and React, following industry-standard practices.
